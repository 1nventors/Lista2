Regra de Cramer - Resolução da Questão 2


Sistema de equações pela regra de Cramer

Explicação da Regra de Cramer

A Regra de Cramer é um método matemático para resolver sistemas lineares, usando determinantes de matrizes do método Ax = B, primeiramente fazemos a determinante da matriz original A, após isso, substituimos cada coluna, uma a uma, pelos valores dos "resultados" B.

Nosso sistema de equações é:

        4x -  y +  z =  8
        2x + 5y + 2z = 3
         x + 2y + 4z = 11

Podemos escrevê-lo na forma matricial Ax = B, onde:

        A = |  4  -1   1  |    B = |  8  |
            |  2   5   2  |        |  3  |
            |  1   2   4  |        | 11  |

1️ Passo: Calcular o determinante de A

O determinante da matriz A, chamado de det(A), é:

        det(A) = 4*(5*4 - 2*2) - (-1)*(2*4 - 2*1) + 1*(2*2 - 5*1)
               = 4*(20 - 4) + 1*(2 - 4) + 1*(4 - 5)
               = 18

2️ Passo: Criar as matrizes substituindo colunas

Criamos três novas matrizes substituindo uma coluna de A pelo vetor B.

Matriz Ax (substitui a primeira coluna por B)

        Ax = |  8  -1   1  |
             |  3   5   2  |
             | 11   2   4  |

O determinante é:

        det(Ax) = 8*(5*4 - 2*2) - (-1)*(3*4 - 2*11) + 1*(3*2 - 5*11)
                = 8*(20 - 4) + 1*(12 - 22) + 1*(6 - 55)
                = -18

Matriz Ay (substitui a segunda coluna por B)

        Ay = |  4   8   1  |
             |  2   3   2  |
             |  1  11   4  |

O determinante é:

        det(Ay) = 4*(3*4 - 2*11) - 8*(2*4 - 2*1) + 1*(2*11 - 3*1)
                = 4*(12 - 22) + 8*(8 - 2) + 1*(22 - 3)
                = 18

Matriz Az (substitui a terceira coluna por B)

        Az = |  4  -1   8  |
             |  2   5   3  |
             |  1   2  11  |

O determinante é:

        det(Az) = 4*(5*11 - 3*2) - (-1)*(2*11 - 3*1) + 8*(2*3 - 5*1)
                = 4*(55 - 6) + 1*(22 - 3) + 8*(6 - 5)
                = 54

3️ Passo: Calcular x, y e z

        x = det(Ax) / det(A) = (-18) / 18 = -1
        y = det(Ay) / det(A) = (18) / 18 = 1
        z = det(Az) / det(A) = (54) / 18 = 3

Portanto, a solução do sistema é (x, y, z) = (-1, 1, 3).

Implementação em C#

Abaixo está o código C# que implementa a Regra de Cramer para resolver esse sistema:


using System;

class Cramer
{
    static void Main()
    {
        double[,] A = { { 4, -1, 1 }, { 2, 5, 2 }, { 1, 2, 4 } };
        double[] B = { 8, 3, 11 };

        double Det(double[,] m) =>
            m[0,0] * (m[1,1] * m[2,2] - m[1,2] * m[2,1]) -
            m[0,1] * (m[1,0] * m[2,2] - m[1,2] * m[2,0]) +
            m[0,2] * (m[1,0] * m[2,1] - m[1,1] * m[2,0]);

        double d = Det(A);
        if (d == 0) { Console.WriteLine("Sistema sem solução única."); return; }

        double[,] Ax = { { B[0], A[0,1], A[0,2] }, { B[1], A[1,1], A[1,2] }, { B[2], A[2,1], A[2,2] } };
        double[,] Ay = { { A[0,0], B[0], A[0,2] }, { A[1,0], B[1], A[1,2] }, { A[2,0], B[2], A[2,2] } };
        double[,] Az = { { A[0,0], A[0,1], B[0] }, { A[1,0], A[1,1], B[1] }, { A[2,0], A[2,1], B[2] } };

        Console.WriteLine($"x1 = {Det(Ax) / d:F2}");
        Console.WriteLine($"x2 = {Det(Ay) / d:F2}");
        Console.WriteLine($"x3 = {Det(Az) / d:F2}");
    }
}
